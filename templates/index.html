{% trans "INTRODUCCIÓN" %}

{% blocktrans %}Los trópicos son conocidos por los altos niveles de biodiversidad que se encuentran en estas áreas, pero se sabe poco sobre la disponibilidad y el uso de la biodiversidad agrícola en los trópicos. Biodiversidad agrícola es la base para una producción agrícola sostenible. Biodiversidad agrícola es también la base para dietas diversas y saludables. El conocimiento sobre el la disponibilidad y uso de biodiversidad agrícola en comunidades agrícolas permite a investigadores, comunitarios, y sus organizaciones de apoyo, para identificar las necesidades y soluciones para mejorar la producción y la comida en estas comunidades agrícolas.{% endblocktrans %}

{% blocktrans %}En este portal, la Red de Datos de Biodiversidad Agrícola comparte conocimiento sobre la disponibilidad y el uso de biodiversidad agrícola en comunidades agrícolas de los trópicos. Hasta el momento la base de datos incluye información de 52 comunidades agrícolas de 8 países. Por lo que sabemos, esta es la base de datos más extensa de la diversidad y uso de biodiversidad agrícola a nivel comunitario en los trópicos.{% endblocktrans %}

{% blocktrans %}Los datos de cada comunidad se recogieron con una metodología común: el análisis de cuatro celdas. Esta metodología es una técnica de evaluación participativa y rápida que fue originalmente creada para evaluar la abundancia y distribución de variedades de cultivos en comunidades agrícolas. La metodología fue adaptada para evaluar la abundancia y uso de biodiversidad agrícola en sistemas alimentarios de comunidades agrícolas. Para cada comunidad agrícola incluida en la base de datos, el portal presenta la producción, consumo, adquisición y comercialización de cada especie en los sistemas alimentarios de comunidades agrícolas.{% endblocktrans %}

{% blocktrans %}Se puede utilizar la información presentada para identificar los vacíos e identificar opciones para la producción sostenible y la diversidad alimentaria en estas comunidades.{% endblocktrans %}
